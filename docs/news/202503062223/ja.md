# 価格更新

サービス向上のため、2025年3月6日より料金を値上げさせていただきます：

価格改定は以下の通りです：

| 名称｜価格｜新価格
| -------- | ------ | --------- |
| タイニー｜2ユーロ｜2.70ユーロ
| スモール｜4ユーロ｜5.10ユーロ｜8ユーロ｜9.70ユーロ
| ミディアム｜8EUR｜9.90EUR
| ラージ｜17 EUR｜21.90 EUR
| X-Lサイズ｜30 EUR｜39.90 EUR
| XX-Lサイズ｜45 EUR｜55.50 EUR

これらの変更は、カスタム価格のリリースと同時に行われます！

### 他にもいくつかの機能をリリースしました：

- ユーザーが設定可能なPTRレコード
- 独立した課金ページ o VM情報ビュー
- VMリソース使用状況グラフ
- 新しいVMにはlnvps.cloudのフォワードDNSエントリーが割り当てられます（例：vm-1.lnvps.cloud）。既存のVMには後日フォワードエントリーが追加されます。
- その他、リソース割り当ての処理に関する多くの小さな改善点があります。
