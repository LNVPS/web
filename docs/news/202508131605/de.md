Eine weitere großartige Funktion, die heute fertiggestellt wurde, ist NWC!

Dies ermöglicht eine automatische Erneuerung direkt über Nostr-Relais, keine manuellen Erneuerungen mehr!

Konfigurieren Sie einfach Ihre NWC-Wallet-Verbindung unter Konto -> Einstellungen

![nwc_setting](https://image.nostr.build/c395b70500aa53fb1fa4ca94a766d117d8268efb2dcc958b4eba053b18466fe0.png)

Aktivieren Sie dann die automatische Erneuerung auf Ihren VM's!

![auto_renew_vm](https://image.nostr.build/11909943d8b9a5e8efeca73a21f7c6c47e4eb52aee7bfcafce7e4bbe29f4833f.png)

Sie können auch für neue VMs bezahlen oder sie manuell über Ihre NWC-Verbindung erneuern.
