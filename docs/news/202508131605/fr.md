Une autre grande fonctionnalité a été mise en place aujourd'hui : la CNO !

Elle permet un renouvellement automatique directement via les relais Nostr, plus de renouvellements manuels !

Il suffit de configurer votre connexion au portefeuille NWC dans Compte -> Paramètres

![nwc_setting](https://image.nostr.build/c395b70500aa53fb1fa4ca94a766d117d8268efb2dcc958b4eba053b18466fe0.png)

Ensuite, activez le renouvellement automatique sur vos VM !

![auto_renew_vm](https://image.nostr.build/11909943d8b9a5e8efeca73a21f7c6c47e4eb52aee7bfcafce7e4bbe29f4833f.png)

Vous pouvez également payer pour de nouvelles VM ou les renouveler manuellement avec votre connexion NWC.

![nwc_payment](https://image.nostr.build/e9bc94ca4b9129ceeab93103c6b17e5e2a22019894373ecacaffd3552c7c07ed.png)
