Nous sommes ravis d'annoncer la nouvelle fonctionnalité de la console Web !

Vous pouvez désormais accéder au terminal de votre VM directement depuis votre navigateur - aucun client SSH n'est nécessaire. C'est parfait pour les tâches de maintenance rapide, le débogage, ou lorsque vous n'êtes pas sur votre poste de travail habituel.

Il vous suffit de naviguer vers votre VM et de cliquer sur le bouton Console pour accéder instantanément au terminal.

Caractéristiques :

- Accès au terminal en temps réel via WebSocket
- Indicateur d'état de la connexion
- Reconnexion en un clic en cas de déconnexion
- Rendu accéléré par WebGL pour des performances fluides

Cette fonctionnalité est particulièrement utile lorsque vous devez résoudre des problèmes de réseau susceptibles d'empêcher l'accès au SSH.

**Note:** Pour utiliser la console web, vous devrez d'abord définir un mot de passe pour votre utilisateur via SSH en exécutant `passwd`.
