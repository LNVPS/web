*이 게시물은 영어에서 자동으로 번역되었습니다.*

새로운 웹 콘솔 기능을 발표하게 되어 기쁩니다!

이제 브라우저에서 직접 가상 머신의 터미널에 액세스할 수 있습니다. SSH 클라이언트는 필요하지 않습니다. 이는 간단한 유지 관리 작업, 디버깅 또는 평소 작업 공간에서 멀리 떨어져 있을 때 유용합니다.

가상 머신으로 이동하여 콘솔 버튼을 클릭하면 즉시 터미널에 액세스할 수 있습니다.

기능:

- WebSocket을 통한 실시간 터미널 액세스
- 연결 상태 표시기
- 연결이 끊어진 경우 한 번의 클릭으로 재연결
- 부드러운 성능을 위한 WebGL 가속 렌더링

이 기능은 SSH 액세스를 방해할 수 있는 네트워크 문제를 해결해야 할 때 특히 유용합니다.

**참고:** 웹 콘솔을 사용하려면 먼저 `passwd`를 실행하여 사용자 계정에 암호를 설정해야 합니다.
