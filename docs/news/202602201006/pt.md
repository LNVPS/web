Temos o prazer de anunciar o novo recurso do Console da Web!

Agora você pode acessar o terminal da sua VM diretamente do seu navegador, sem necessidade de cliente SSH. Isso é perfeito para tarefas rápidas de manutenção, depuração ou quando você estiver longe de sua estação de trabalho habitual.

Basta navegar até sua VM e clicar no botão Console para obter acesso instantâneo ao terminal.

Recursos:

- Acesso ao terminal em tempo real via WebSocket
- Indicador de status da conexão
- Reconexão com um clique em caso de desconexão
- Renderização acelerada por WebGL para um desempenho suave

Esse recurso é especialmente útil quando você precisa solucionar problemas de rede que podem impedir o acesso SSH.

**Observação:** Para usar o console da Web, primeiro será necessário definir uma senha para o seu usuário via SSH executando `passwd`.
