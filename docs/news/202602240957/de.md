Verdienen Sie Sats, indem Sie LNVPS weiterempfehlen! Wir haben ein Empfehlungsprogramm gestartet, das Sie bezahlt, wenn jemand eine neue VM über Ihren Empfehlungslink bestellt.

## Wie es funktioniert

1. Gehen Sie zu **Konto -> Empfehlungsprogramm** und melden Sie sich an
2. Teilen Sie Ihren einzigartigen Empfehlungslink mit anderen
3. Wenn jemand Ihren Link besucht und eine neue VM bestellt, erhalten Sie eine Auszahlung

## Auszahlungsoptionen

Wählen Sie, wie Sie bezahlt werden möchten:

- **Lightning Adresse** - Auszahlungen werden direkt an eine beliebige Lightning Adresse gesendet
- **NWC Wallet** - Auszahlungen werden über Ihre konfigurierte Nostr Wallet Connect Wallet gesendet

Sie können auf der Seite mit den Empfehlungseinstellungen jederzeit zwischen den Auszahlungsmethoden wechseln.

## Verfolgen Sie Ihren Verdienst

Ihr Referral-Dashboard zeigt alles auf einen Blick: erfolgreiche Referrals, nach Währungen aufgeschlüsselte Verdienste und einen vollständigen Auszahlungsverlauf.

Starten Sie noch heute unter **Konto -> Empfehlungsprogramm** und beginnen Sie zu verdienen!
