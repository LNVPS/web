*이 게시물은 영어에서 자동으로 번역되었습니다.*

추천 프로그램을 시작한 이후 꾸준히 서비스를 제공하고 있습니다. 그 이후 제어판에 추가된 모든 기능은 다음과 같습니다.

## 새로운 테마와 재설계된 UI

두 가지 새로운 테마인 **네온** 다크 테마와 **라이트** 테마로 인터페이스가 완전히 변경되었습니다. 내비게이션 바에 있는 새로운 테마 전환기를 사용하여 언제든지 테마를 전환할 수 있습니다. VM 목록은 이제 서버를 한눈에 더 많은 정보를 볼 수 있는 간결한 행 레이아웃으로 표시합니다.

## Revolut 카드 결제

비트코인 라이트닝 및 NWC 외에 이제 임베디드 Revolut 결제 시스템을 통해 **Revolut 카드**로 결제할 수 있습니다. 결제 프로세스가 재설계되어 사용 가능한 모든 결제 방법(처리 수수료 포함)을 함께 표시하므로 확인하기 전에 정확히 얼마를 지불해야 하는지 알 수 있습니다.

## 유연한 갱신 간격

VM을 갱신하거나 주문할 때 이제 단일 기간으로 제한되지 않고 여러 결제 간격 중에서 선택할 수 있습니다. 필요에 맞는 간격을 선택하면 가격이 자동으로 업데이트됩니다.

## 문의 페이지

일반 문의를 위한 새로운 **문의** 페이지가 주 내비게이션에서 제공됩니다.

## IP 주소 목록

할당된 IP 주소 및 서브넷이 계정 개요에 표시되므로 계정과 관련된 모든 리소스를 한곳에서 쉽게 확인할 수 있습니다.

## CPU 정보

VM 카드 및 리소스 페이지는 이제 CPU 제조업체, 아키텍처 및 기능 플래그를 표시하여 VM이 실행되는 하드웨어에 대한 더 명확한 정보를 제공합니다.

## 계정 페이지 재설계

계정 섹션이 **지속적인 사이드바** 내비게이션으로 재구성되었습니다. 각 섹션(설정, SSH 키, 도메인, 추천, 메시지, 지원)에는 이제 명확한 레이블과 활성 상태 표시기가 있는 자체 페이지가 있습니다.

## Nostr 도메인 경로 프록시

Nostr 도메인 구성 페이지에는 이제 직접 하위 도메인 구성의 대안으로 **경로 기반 프록시** (`/_nostr/...`)를 설정하는 단계별 지침이 포함됩니다.

## 암호화된 직접 메시지(NIP-17)

지원팀은 NIP-17 표준을 사용하여 종단 간 암호화된 직접 메시지를 통해 통신합니다. 계정 섹션의 전용 **메시지** 탭에서 지원팀의 메시지를 보고 답장할 수 있습니다. 메시지는 로컬에서 Nostr 키를 사용하여 복호화되고 브라우저에 캐시되어 빠르게 로드됩니다.

## 주문 시 이메일 확인

새 VM 주문 시 계정 이메일이 아직 확인되지 않은 경우 인라인 이메일 확인 단계가 나타납니다. 이렇게 하면 주문 프로세스가 한 곳에서 진행되어 다른 페이지로 리디렉션되지 않습니다.

## VM 세부 정보 페이지에서 재설치

재설치 작업이 VM 목록에서 각 **VM 세부 정보 페이지**로 이동하여 다른 관리 작업과 함께 사용할 수 있습니다.

## 다국어 지원

이제 전체 제어판을 아랍어, 중국어, 독일어, 스페인어, 프랑스어, 일본어, 한국어, 포르투갈어, 러시아어, 튀르키예어의 10개 언어로 사용할 수 있습니다. 브라우저의 기본 언어가 자동으로 감지되며 내비게이션 바의 언어 선택기에서 언제든지 전환할 수 있습니다. 날짜, 숫자 및 비용은 선택한 로케일에 맞게 형식이 지정되며 아랍어는 전체적으로 오른쪽에서 왼쪽으로 레이아웃으로 표시됩니다.
