3月に前回お伝えして以来、コントロールパネルにさまざまな改善を加え、継続的にアップデートを行ってきました。今回、新たに追加された機能をご紹介します。

## VMごとのファイアウォールルール

 теперь вы можете управлять фаерволом для каждого VM непосредственно из панели управления. Странице VM добавлена новая вкладка «Firewall», позволяющая задавать политики по умолчанию для входящего и исходящего трафика, а также добавлять до 20 пользовательских правил. Каждое правило поддерживает направление, протокол (TCP/UDP/ICMP), порт или диапазон портов, адрес источника/назначения и действие (разрешить, отклонить или запретить). Правила можно включать, редактировать или удалять в любое время, что обеспечивает детальный контроль над сетевым доступом без необходимости внесения изменений в гостевую ОС.

## TelegramとWhatsAppによる通知

Eメールと暗号化されたNostr DMに加えて、 теперь вы можете получать уведомления об учетной записи через Telegram и WhatsApp. Управляйте всеми каналами уведомлений на обновленной странице настроек учетной записи, где для каждого канала отображается текущий статус подключения, и каждый канал можно включать или отключать независимо.

## アカウント設定の再設計

Страница настроек учетной записи была реорганизована в четкие секционированные панели с строками состояния в стиле терминала для каждого канала уведомлений. На вкладке «Поддержка» больше не требуется вручную вставлять ваш открытый ключ — теперь он включается автоматически, а на вкладке «Сообщения» хранятся только для чтения прошлые сообщения службы поддержки.

## サイドバーナビゲーションを備えたVM概要

Страница сведений о VM имеет новую компоновку с боковой панелью, с отдельными разделами для «Обзор», «Выставление счетов», «Консоль», «Брандмауэр», «Графики», «История» и «Обновление». Страница обзора была переработана с четкой сеткой спецификаций (ЦП, ОЗУ, диск, ОС, регион, ключ SSH), динамическим индикатором состояния, отображающим загрузку ЦП и ОЗУ, и информацией об истечении срока действия с быстрыми действиями «Оплатить сейчас» / «Продлить». Для VM, которые все еще находятся в процессе подготовки, теперь отображается анимированное изображение загрузки вместо неполной информации, а для совершенно новых VM, ожидающих первой оплаты, четко отображается соответствующий значок.

## Улучшения страницы состояния

На публичной странице состояния теперь отображается **ежемесячный график доступности** за последние 12 месяцев, что позволяет быстро увидеть историческую доступность. Мы также добавили состояние **«Запланировано»** для плановых работ, что позволяет четко определить сроки выполнения, а не непредвиденное возникновение проблемы.
